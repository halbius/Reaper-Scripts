This Script links Markers to items.
When moving the item, the marker follows. Moving the marker, the item follows.

<h1>Requirement</h1>: 

The markername has to start with '§'

<h1>Installation:</h1>

1. Put both Scripts in Reapers Resources Script folder.
2. Put the ca_markeritemlink_btn.lua in the toolbar
3. If you like, put the toolbar.png's in .../Reaper/data/toolbar_icons and use them for the toolbar button.

<h1>Usage:</h1>

<h2>1. Button:</h2>
- Click opens the Script window and starts the script if it is not running
- Shift-Click terminates the script
- Alt-Click toggles Sync on-off

<h2>2. Linking:</h2>
Any marker starting with § ist listed in the scripts window.
Per default auto link is on. When dragging an item over the next correctly named (§) marker, they are linked.
If you turn auto link off, you can link the marker manualy to an item by clicking on 'link' in the markers row.
Any couple can be unlinkend individually, or all together.
